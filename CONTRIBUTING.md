# Contributing

Improvements to the SKILL.md prompt are welcome — open a PR with before/after examples showing the change.

## How

1. Fork repo
2. Edit `skills/soulmate/SKILL.md` — this is the only copy you need to touch
3. Open PR with:
   - **Before:** what soulmate says now
   - **After:** what soulmate says with your change
   - One sentence why the change is better

> **Note:** `soulmate/SKILL.md`, `plugins/soulmate/skills/soulmate/SKILL.md`, and `soulmate.skill` are auto-synced by CI after merge. Do not edit them directly.

Small focused changes are welcome. Big rewrites need discussion first.
